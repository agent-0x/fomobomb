# FomoBomb — Axon 链上 FOMO 博弈游戏

> **核心规则：每次投注重置 100 个区块（约 10 分钟）的倒计时。连续 10 分钟没人投注 → 游戏结束 → 最后一个投注的人赢走所有奖池。**

每次投注还有 **2% 概率** 命中跨局累积的随机大奖。

## 一句话理解

你投 1 AXON 进去，倒计时重置。别人也投，倒计时又重置。没人敢投了？等 10 分钟（100 个区块），最后那个人通吃。但谁都不想当"倒数第二个"。

## 钱怎么分

每次投注扣 5% 手续费，剩下 95% 进奖池：

```
你投 1 AXON
  └─ 0.95 → 当局奖池（最后一人赢走）
  └─ 0.03 → 大奖池（2% 概率即开即中，跨局累积）
  └─ 0.02 → 金库
```

## 博弈点

**核心矛盾：每个人都想当最后一个，但如果大家都等着不投，倒计时就走完了。**

- 你投了，别人可能跟投把你挤掉
- 你不投，万一没人投了你就错过了
- 倒计时越短，心理压力越大
- 大奖池越大，越多人忍不住"赌一把"
